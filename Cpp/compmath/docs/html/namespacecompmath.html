int compmath::export_data_to_csv ( const std::string & filename,
const std::vector< T > & x,
const std::vector< T > & func )

Exports data to a CSV file.

This function writes the provided x-coordinates and corresponding function values to a CSV file. The output file will include a header row and the data in two columns: "x" and "value".

Template Parameters
TThe type of the elements in the vectors. Must support streaming to an output stream.
Parameters
filenameThe name of the CSV file to create (relative to EXPORT_PATH).
xA vector containing the x-coordinates.
funcA vector containing the corresponding function values.
Returns
int Returns 0 on success, or -1 if the file could not be opened.
Exceptions
std::invalid_argumentIf the sizes of x and func do not match.
std::runtime_errorIf the directory specified by EXPORT_PATH cannot be created.
Note
The directory specified by EXPORT_PATH must exist or be creatable. Ensure that the type T supports the << operator for output streams.

template<typename T, typename Func, typename Integrator>
std::pair< T, T > compmath::runge_rule ( Integrator integrator,
Func func,
T a,
T b,
size_t n,
int order )

Estimate the integral and its error using Runge's rule.

This function computes the integral of a function using a specified integration method with two different numbers of intervals (n1 and n2), and estimates the error using Runge's rule.

Template Parameters
TNumeric type (e.g., double, float)
FuncCallable type: T func(T)
IntegratorCallable type for integration method: T integrator(Func, T, T, size_t)
Parameters
integratorIntegration method (e.g., integrate_simpson, integrate_trapez, integrate_rect)
funcFunction to integrate
aLower bound of integration (specify type! ex: 3.0, not 3)
bUpper bound of integration (specify type! ex: 3.0, not 3)
nNumber of intervals for the first estimate (result estimate will use 2*n)
orderThe order of the integration method (e.g., 2 for trapezoidal, 4 for Simpson)
Returns
std::pair<T, T> where first is the refined integral estimate (using n2), and second is the estimated error
Note
Example usage:
auto [I2n, error] = compmath::runge_rule(
compmath::integrate_simpson<double, decltype(f)>, f, 0.0, M_PI, 10, 4
);
